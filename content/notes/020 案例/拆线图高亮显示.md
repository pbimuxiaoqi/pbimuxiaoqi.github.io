```js
Selected Ticker Close = 
VAR SelectedTicker = SELECTEDVALUE(Tickers[Ticker])
RETURN
CALCULATE(
    SUM(Stocks[Close]),
    Stocks[Ticker] = SelectedTicker
)
```

新建一个计算表,不需要和事实表建立关系

```js
Tickers = 
VALUES(Stocks[Ticker])
```
