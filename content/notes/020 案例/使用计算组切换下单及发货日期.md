销售记录中通常会有多个日期，比如下单日期，发货日期，想根据这些日期来查看数据时，往往需要再写度量值，激活虚拟关系。
可以使用计算组来实现日期的切换 
新建一个计算组，创建一个计算项Order Date, 数据模型中和日期表是实体关系 
```js
SELECTEDMEASURE ()
```
再创建一个计算项
```js
CALCULATE ( 
    SELECTEDMEASURE (),
    USERELATIONSHIP ( Sales[Delivery Date], 'Date'[Date] )
)
```

基础度量如下，不需要特意改动
```js
Sales Amount =
SUMX (
	Sales,
	Sales[Quantity] * Sales[Net Price]
)
```
